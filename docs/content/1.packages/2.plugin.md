## Rules

### `nuxt/prefer-import-meta`

This rule enforces the use of `import.meta.dev` / `import.meta.server` in Nuxt 3 projects instead of `process.client` / `process.server`.

```json
{
  "rules": {
    "nuxt/prefer-import-meta": "error"
  }
}
```
